📸 forpie 웹사이트 사진 넣는 방법
=====================================

이 폴더(images)에 아래 이름으로 사진을 저장하면 웹사이트에 자동으로 나타납니다.
(파일 이름과 확장자 .jpg 를 똑같이 맞춰주세요)

[ 필수 사진 ]
  hero.jpg        →  메인 첫 화면 (가로로 넓은 사진)
                     추천: 카페 외관 + 초록 풍경, 또는 꽃과 파이가 함께 있는 분위기 컷
  story-1.jpg     →  소개 섹션 사진 (세로 4:5)
                     추천: 창가의 계절 꽃 + 식물, 또는 파이 만드는 손

[ 갤러리 사진 6장 ]
  gallery-1.jpg   →  세로형  (예: 아침 햇살 드는 자리)
  gallery-2.jpg   →  정사각형(예: 튤립/거베라 등 오늘의 꽃)
  gallery-3.jpg   →  세로형  (예: 핸드메이드 파이 클로즈업)
  gallery-4.jpg   →  가로형  (예: 브런치 테이블 세팅)
  gallery-5.jpg   →  세로형  (예: 음료 / 디저트)
  gallery-6.jpg   →  정사각형(예: 초록 식물 코너 / 인테리어)

[ 선택 사진 ]
  location.jpg    →  찾아오시는 길의 지도 자리 (외관 사진 추천)

-------------------------------------
✅ 사진이 없어도 초록·꽃 톤 배경으로 예쁘게 보입니다.
✅ 사진을 넣은 뒤 브라우저에서 새로고침(F5)하면 바로 반영돼요.
✅ 사진 이름 바꾸기가 번거로우면, 사진을 이 폴더에 그냥 넣고
   "사진 넣었어요"라고 말씀해 주시면 제가 자동으로 연결해 드립니다.

* 인스타그램(@forpie_cafe)의 꽃·파이·인테리어 사진들을 그대로 쓰면
  사이트 분위기가 인스타와 완벽하게 통일됩니다.

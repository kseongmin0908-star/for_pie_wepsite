# forpie · 단양 브런치 카페 웹사이트 🥧🌿

> 덴마크에서 만난 파이 한 조각이, 단양의 작은 시골 카페가 되었습니다.

단양 브런치 카페 **forpie(폴파이)** 의 원페이지 홈페이지입니다.
따뜻한 보태니컬 가든 감성(크림 + 세이지 그린 + 다채로운 꽃)으로 제작했어요.

## 미리보기
`index.html` 파일을 브라우저로 열면 바로 확인할 수 있습니다.

## 구성
- **메인 / 소개 / 메뉴 / 갤러리 / 찾아오시는 길** 5개 섹션
- 떨어지는 꽃잎 애니메이션, 스크롤 등장 효과
- 사진은 `images/` 폴더에서 관리 (같은 이름으로 덮어쓰면 교체됩니다)
